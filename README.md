# BロールMP4メーカー

GPTsなどで作ったBロールスライド画像をアップロードし、FISH Audioのナレーション付きMP4を書き出す共有用Webアプリです。

## できること

- PNG/JPG/WebP、または画像入りZIPをアップロード
- 台本テキストからFISH Audioでナレーション生成
- 音声の長さに合わせてスライド表示時間を自動調整
- 縦9:16、横16:9、正方形1:1に対応
- VREW字幕用に、動画下部の余白を0〜30%で確保
- 白文字＋太い黒縁取りの字幕焼き込みに対応
- 完成MP4をダウンロード
- `APP_PASSWORD` で簡易パスワード保護

## Renderで使う手順

1. このフォルダをGitHubリポジトリにアップロードします。
2. Renderで「New Web Service」を作成し、このリポジトリを選びます。
3. 環境変数を設定します。

```text
FISH_AUDIO_API_KEY=FISH AudioのAPIキー
FISH_AUDIO_VOICE_ID=使いたい音声モデルID
FISH_AUDIO_MODEL=s2-pro
APP_PASSWORD=受講者に渡すログインパスワード
FLASK_SECRET_KEY=長めのランダム文字列
MAX_UPLOAD_MB=300
```

4. デプロイ後、発行されたURLを特典ページから案内します。

## 使い方

1. パスワードでログインします。
2. スライド画像、または画像入りZIPをアップロードします。
3. ナレーション台本を貼り付けます。
4. 画面サイズ、スライドの合わせ方、VREW字幕スペースを選びます。
5. 「MP4を作成」を押します。
6. 完成後にMP4をダウンロードします。

## スライド画像の並び順

ファイル名順に並びます。

```text
01.png
02.png
03.png
```

のように番号を付けてください。

## 注意

- FISH AudioのAPI利用料は、設定したAPIキーのアカウントに発生します。
- 広く配る場合は、Renderのプラン、同時利用数、API利用上限を確認してください。
- 公開配布する場合は、`APP_PASSWORD` を必ず設定してください。
- 生成済みファイルはサーバー内の `jobs/` に保存されます。長期運用では定期削除の仕組みを追加してください。
